# Zepto SQL Data Analysis

## Project Overview

This project analyzes Zepto's product dataset using PostgreSQL. The objective is to explore product pricing, discounts, stock availability, and category-wise trends by solving real-world business questions with SQL.


## Dataset

The dataset contains product-level information, including:

* Product Name
* Category
* MRP
* Selling Price
* Discount Percentage
* Available Quantity
* Stock Status
* Weight


## Tools & Technologies

* PostgreSQL
* SQL


## Project Workflow

* Imported the dataset into PostgreSQL.
* Cleaned and prepared the data for analysis.
* Performed exploratory data analysis (EDA).
* Wrote SQL queries to answer business questions.
* Generated insights from the data.


## Business Questions Solved

* Find the top 10 products with the highest discounts.
* Identify out-of-stock products.
* Compare MRP and selling price.
* Calculate the average selling price by category.
* Count the number of products in each category.
* Find products with the highest MRP.
* Analyze discount trends across categories.
* Generate category-wise product insights.
